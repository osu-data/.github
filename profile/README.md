<center> 
Welcome to **osu!data**, a community-driven initiative dedicated to providing structured, comprehensive, and up-to-date datasets for the rhythm game *osu!*.

## 📊 About the Project

This organization hosts a collection of repositories containing detailed metadata 
and parameters for every existing beatmap in the game. To ensure accessibility 
and ease of use, data is strictly categorized by game mode. Whether you are building 
a recommendation engine, performing statistical analysis, or developing third-party 
tools, our goal is to provide the raw information you need.

---

## 🗂️ Repositories

Each game mode is maintained in its own dedicated repository:

* **atlas-circles:** Full dataset for the classic circle-clicking mode.
* **atlas-taiko:** Comprehensive data for Taiko maps, including difficulty metrics and metadata.
* **atlas-catch:** Detailed information on all Catch the Beat maps.
* **atlas-mania:** Structured data for all Mania key counts and patterns.

---

## ⚙️ Data Included

The datasets include various parameters such as:

* Beatmap and Set IDs
* File hashes (MD5 for maps and SHA256 for other resources)
